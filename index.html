Local Server Required

YouTube blocks embeds from file:// origins (Error 153). This page needs to run from a web server. It's a one-liner:

1. Open Terminal / Command Prompt
2. cd to the folder containing this file
3. Run one of these commands:
python3 -m http.server 8080
click to select · then Cmd/Ctrl+C to copy

Then open: localhost:8080

Alternatives: npx serve · php -S localhost:8080 · VS Code Live Server
